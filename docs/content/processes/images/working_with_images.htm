Working with Images

Summary:

Describes the process to add images in projects.

Solution:

Inserting Images

Projects use thumbnails for most images, see CompConnect, HR Portals, TRSS, and WAPS for image-specific information.

  1. Create a new paragraph for the image.
  2. Click the Insert tab, and then click Image.
  3. Navigate to the image that you want to insert.
  4. On the General tab of the Insert Image window, click Style Class, and then select the percentage for the thumbnail.
  5. There are no set standards for screen tips and alternate text.
  6. Click OK.
