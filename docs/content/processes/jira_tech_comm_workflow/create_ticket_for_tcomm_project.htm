Creating a JIRA Ticket for a TechComm Project

Summary:

Describes steps to create a ticket for TechComm and deviations for CompConnect and TRSS tickets.

Solution:

To create a ticket:

  1. In Jira, click Create Issue.
  2. In the Project list, select Technical Communication.
  3. In the Issue Type list, select Minor Enhancement.
  4. Complete the following fields:
    • Summary - Enter the headline of the ticket.
    • Description - Provide more details about the ticket.
    • Affects Version - Select the project for the ticket (e.g., TRSS, WAPS, CompConnect, HR Portals, etc.).
    • Documentation Lead - Enter the name of the person responsible for the work, if known.
    • Components - Select the deliverables associated with the task.
    • Fix Version - Select the planned release for the content.
    • Sprint - If known, you can select the sprint for the ticket here.
  5. Click Create.

You can see the ticket in the backlog of the project:

  • CompConnect
  • TRSS
  • WAPS
  • IAP
  • HR Portal
