JIRA Dashboards and Filters

Summary:

Lists the JIRA Dashboards and Filters utilized by TechComm.

Full Description:

You can view and copy some basic dashboards and filters that provide information about tickets in the TechComm JIRA project from the following links:

• TComm JIRA Workflow Writer Dashboard

• TComm JIRA Project PA and SME Dashboard

The following provides a more detailed description with examples of correct and incorrect usage, if applicable.

Title Purpose For Query
TCOMM - Help Content Ready for PA Review This filter shows tickets that are ready for PA review. PA project = "Technical Communication" AND status = "Development Pending Peer Review" AND "PA Lead" = currentUser() AND "QA Approver" is not EMPTY AND JAMA is not EMPTY AND "QA Approver" != "jama.revision" and "PA Approver" is EMPTY
TCOMM - Help Content Ready for PA Review - Test This filter is a test of the TCOMM - Help Content Ready for PA Review filter where the "PA Lead" = currentUser() is replaced by assignee = currentUser(). TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" AND assignee = currentUser() AND "QA Approver" is not EMPTY AND JAMA is not EMPTY AND "QA Approver" != "jama.revision" and "PA Approver" is EMPTY
TCOMM - Help Content in PA Review This filter shows all help content that is in PA review. TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" AND "PA Lead" is not EMPTY AND "QA Approver" is not EMPTY AND JAMA is not EMPTY AND "PA Approver" is EMPTY AND "QA Approver" != "jama.revision"
TCOMM - My Help Content in PA Review This filter shows help content assigned to the current user that is in PA review. TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" AND assignee = currentUser() and "PA Lead" is not EMPTY AND "QA Approver" is not EMPTY AND JAMA is not EMPTY AND "PA Approver" is EMPTY AND "QA Approver" != "jama.revision"
TCOMM - Help Content PA Review Pipeline This filter shows help content currently undergoing peer review. It shows tickets assigned to a peer reviewer as indicated by the PeerName label and any tickets assigned to the writer for revision. PA project = "Technical Communication" AND status = "Development Pending Peer Review" AND labels != Peer AND "QA Approver" is EMPTY or project = "Technical Communication" AND status = "Development Pending Peer Review" AND labels != Peer AND "QA Approver" = "jama.revision"
TCOMM - Help Content PA Review Pipeline - Test This filter is a test of the Help Content PA Review Pipeline filter where the "PA Lead" = currentUser() is replaced by assignee = currentUser(). TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" AND assignee = currentUser() AND labels != Peer AND "QA Approver" is EMPTY OR project = "Technical Communication" AND status = "Development Pending Peer Review" AND assignee = currentUser() AND labels != Peer AND "QA Approver" = "jama.revision"
TCOMM - Help Content Ready for SME Review This filter shows tickets that are ready for SME review. SME project = "Technical Communication" AND status = "Development Pending Peer Review" and "Approver" = currentUser() and "PA Approver" is not EMPTY AND "PA Approver" != "jama.revision" AND JAMA is not EMPTY AND "SME Approver" is EMPTY
TCOMM - Help Content Ready for SME Review - Test This filter is a test of the Help Content Ready for SME Review filter where the "Approver" = currentUser() is replaced by assignee = currentUser(). TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" and assignee = currentUser() and "PA Approver" is not EMPTY AND "PA Approver" != "jama.revision" AND JAMA is not EMPTY AND "SME Approver" is EMPTY
TCOMM - Help Content in SME Review This filter shows all help content that is in SME review. TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" AND Approver is not EMPTY AND "PA Approver" is not EMPTY AND "PA Approver" != "jama.revision" AND JAMA is not EMPTY AND "SME Approver" is EMPTY
TCOMM - My Help Content in SME Review This filter shows help content assigned to the current user that is in SME review. TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" and assignee = currentUser() AND Approver is not EMPTY AND "PA Approver" is not EMPTY AND "PA Approver" != "jama.revision" AND JAMA is not EMPTY AND "SME Approver" is EMPTY

TCOMM - Help Content in SME Review Pipeline

 This filter shows help content currently undergoing PA review. It shows tickets assigned to a PA reviewer as indicated by the completion of peer review and inclusion of a JAMA link. It also shows any tickets requested for revision by the PA. SME project = "Technical Communication" AND status = "Development Pending Peer Review" AND "PA Lead" is not EMPTY AND "QA Approver" is not EMPTY AND JAMA is not EMPTY AND "PA Approver" is EMPTY AND "QA Approver" != "jama.revision" OR project = "Technical Communication" AND status = "Development Pending Peer Review" AND "PA Lead" is not EMPTY AND "QA Approver" is not EMPTY AND JAMA is not EMPTY AND "PA Approver" = "jama.revision"
TCOMM - Help Content SME Review Pipeline Test This filter is a test of the Help Content SME Review Pipeline filter where the "Approver" = currentUser() is replaced by assignee = currentUser(). TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" AND assignee = currentUser() AND "PA Lead" is not EMPTY AND "QA Approver" is not EMPTY AND JAMA is not EMPTY AND "PA Approver" is EMPTY AND "QA Approver" != "jama.revision" OR project = "Technical Communication" AND status = "Development Pending Peer Review" AND assignee = currentUser() AND "PA Lead" is not EMPTY AND "QA Approver" is not EMPTY AND JAMA is not EMPTY AND "PA Approver" = "jama.revision"
TCOMM - Help Content Ready for Peer Review This filter shows tickets that are ready for peer review, but no review has started editing the content. TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" and "QA Approver" is EMPTY

TCOMM - My Help Content to Peer Review

 This filter is a template to create personal filters based on the label that you use when you perform a peer edit. Change the value of PeerName to your label, or use one personal ones already available. TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" and "QA Approver" is EMPTY and labels = PeerName

 Shows the peer reviews that Dave is working. TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" and "QA Approver" is EMPTY and labels = PeerDave

TCOMM - My Help Content to Prep for JAMA Review Shows content assigned to a specific writer that has passed peer review. The writer needs to prepare the JAMA review so that the ticket appears on the PA's dashboard. TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" AND assignee = currentUser() AND "QA Approver" is not EMPTY AND JAMA is EMPTY
TCOMM - Help Content to Prep for JAMA Review Shows content assigned to all writers that has passed peer review. The writer needs to prepare the JAMA review so that the ticket appears on the PA's dashboard. TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" and "QA Approver" is not EMPTY and JAMA is EMPTY
TCOMM - My Help Content that Needs Revisions Shows content assigned to a specific writer that requires revision as specified by the jama.revision user in any of the QA, PA, or SME Approver fields. TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" AND assignee = currentUser() and "QA Approver" = "jama.revision" OR status = "Development Pending Peer Review" and "PA Approver" = "jama.revision" OR status = "Development Pending Peer Review" and "SME Approver" = "jama.revision"
TCOMM - Help Content that Needs Revisions Shows content assigned to all writers that requires revision as specified by the jama.revision user in any of the QA, PA, or SME Approver fields. TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" and "QA Approver" = "jama.revision" OR status = "Development Pending Peer Review" and "PA Approver" = "jama.revision" OR status = "Development Pending Peer Review" and "SME Approver" = "jama.revision"
TCOMM - Help Content Ready to Build for QA Shows approved content assigned to all writers that needs to be moved to the Ready to Build for QA status. TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" and "SME Approver" is not EMPTY AND "SME Approver" != "jama.revision" AND "PA Approver" is not EMPTY and "PA Approver" != "jama.revision"
TCOMM - My Help Content Ready to Build for QA Shows approved content assigned to a writer that needs to be moved to the Ready to Build for QA status. TCOMM project = "Technical Communication" AND status = "Development Pending Peer Review" AND assignee = currentUser() and "SME Approver" is not EMPTY AND "SME Approver" != "jama.revision" AND "PA Approver" is not EMPTY and "PA Approver" != "jama.revision"
