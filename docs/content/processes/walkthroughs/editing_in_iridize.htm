Editing in Iridize

Full Description:

Iridize continues to update and expand the functionality offered when editing a guide. As the functionality changes, this page should be updated to reflect those changes.

Requirements

You must use Chrome to record tutorials and you must use their extension. You'll be prompted for it when you go to https://iridize.com/account/manage/.

Terminology

  • Target
  • Prod & Dev

  • Tutorial synonyms: guide, walkthrough, interactive walkthrough.
  • Step synonyms: tip, ...

Editing in HTML

See the Iridize Editor Manual for detailed steps, but review the following guidance to ensure consistency.

Use the internal styles.

  • Normal: Use for all paragraphs. You must apply this manually to each paragraph.
  • Heading 1: 1st tip only, title of the tutorial
  • Heading 2: Congratulations page and any step-specific titles
  • Heading 3: Strong messaging or if you really need a heading on a step page after a H2

When in doubt, edit in source

Click Source to view the underlying HTML. After you complete edits in here, click Source again. If you forget this step, you'll lose all of your changes.

Interactive Tutorials

Whenever you have a interactive tutorial, add this line to the first splash tip:

  • CompConnect: <img alt="" height="25" src="https://helpuat.ehr.com/TC_Images/touchpoint-512.png" width="25" /><strong>When you see this icon, you must interact with the application to advance</strong>.
  • WAPS: <img alt="" src="https://waps.ehr.com/TCommfiles/touchpoint-512_white.png" /> <strong>This tutorial is interactive.</strong>

NOTE Use a space between icons and text.

Create Images

  1. Use SnagIt to take the screen shot.
  2. Crop and resize the image as needed.
  3. Save as a PNG to:
    • CompConnect:\\TRHIIS-NA2-01U\Deploy\Help\TC_Images (link = https://helpuat.ehr.com/TC_Images/image_name.png)
    • WAPS: \\HRTIIS-NA2-01Q\TCommFiles\Images and gifs (link = https://waps.ehr.com/TCommfiles/Images and gifs/image_name.png.
  4. Copy the path and filename for the image.
  5. In the step window of Iridize, position the cursor in the tip and click the link icon in the local toolbar.
  6. Paste the image link and click OK.

Create GIFs

Sometimes GIFs are used instead of complex interactions.

  1. Animated videos must be converted to GIF before they can be inserted into Iridize steps.
  2. In Snagit, capture video (no more than 10 seconds).
  3. Save as a MP4.
  4. Go to http://ezgif.com/
  5. Click Video to GIF.
  6. Upload your MP4.
  7. Preview it and change the end time (to match the actual end time of the video).
  8. Convert it to GIF.
  9. Download it.
  10. Post to https://waps.ehr.com/TCommfiles/videoimage_name.gif.
  11. Copy the path and filename for the GIF.
  12. In the step window of Iridize, position the cursor in the tip and click the link icon in the local toolbar.

    13. NOTE Even though it is an animation, it is inserted in Iridize as an image.

  13. Paste the video image link and click OK.

Creating Hyperlinks to Tutorials

At the end of a hyperlink, add ?stStart=API_Name, see the example below. If you're logged into CompConnect QA, this link should take you straight to a tutorial.

http://cocoqa.us.datacenters.ww/Jobs/tenant/DEMO25/Job/NewJobGrid/?stStart=gw3myp7r
