Creating New Topics

Summary:

Create new topics in Flare and add them to source control (TFS).

Full Description:

Flare's help documentation covers much of this procedure. Rather than providing all of the details, this procedure covers the information that you should start with and add the new topic to TFS.

To create a topic:

  1. Create the topic in an existing folder.
    • Add a new topic and enter the topic properties.
    • Copy an existing file and change the topic properties. Many of the help projects have a "latest" topic file to reuse or a template file.

    Note: If you leave the topic heading blank, Flare uses the file name.

  2. Enter the file name. See File Naming.
  3. Check the topic into source control. This adds the file to TFS.
  4. Set the file tags. For the TRSS project, see Using File Tags.
