Using Snippets for Numbered Lists

Summary:

This describes a method for creating snippets for numbered lists.

Full Description:

One of the problems with using snippets in numbered lists is the numbering. If you create a two-step list and inject it in an existing list, the numbering will get jacked up. One way to get around this is to create the numbered snippet as a regular paragraphs, and then insert it into a numbered list. This doesn't always work though.

Alternatively, you can manage the number continuation in the topic or the snippet, as appropriate.

Append either step_ or steps_ to the snippet file name.

Reuse the Same First Step

You might write several topics that include the same first step. For example, to add, edit or delete pay types and pay sets, you always need to access the Pay Types page. The first step in all of these topics will be "Click Settings, and then select Pay Types." The easiest way to set up this kind of snippet is as a simple paragraph, not as an ordered list. The alternative is to create the snippet as an ordered list, insert it into the <p> tag in a topic, and then follow that up with an ordered list.

To reuse the same first step:

  1. Create a new snippet with a file name that begins with step_.
  2. Enter the text of the step in the <p> tag.
  3. Save the snippet.
  4. In the topic where you insert the snippet, create an ordered list.
  5. Insert the snippet in the first <li> tag.

Reusing Multiple Steps

You might write multiple topics that include multiple steps. For example, reordering grids in Wasabi is done in a functionally universal modal called Manage Order. However, there are multiple topics about reordering specific grid items. These topics might have different first steps, but the last steps are the same. The key to making this work is to use the Continue Sequence setting to make sure that the numbering is correct.

To reuse multiple steps:

  1. Create a new snippet with a file name that begins with steps_.
  2. Create an ordered list.
  3. Enter the text of the steps in the ordered list.
  4. With your cursor in any list item, click the List Actions button, and then select Continue Sequence.
  5. Save the snippet.
  6. In the topic where you insert the snippet, create an ordered list.
  7. When you get to the list item where you want to insert the snippet, end the list.
  8. Insert the snippet in the <p> tag following the last list item.
