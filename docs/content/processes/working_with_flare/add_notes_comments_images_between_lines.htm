Adding Notes, Comments, Images between Lines

Summary:

Use these steps to insert notes, comments, or images between items in a list by placing <p> tags after the <li> tags (without using the editor)!

Full Description:

  1. Start a numbered list, type a line of content, and press Enter.
  2. Type Ctrl + ; (Ctrl and semi-colon).
  3. Type the next line of content.
  4. Press Enter. This icon appears,
  5. Continue typing the next line of content and press Enter.
  6. To restart numbering, click the icon.
  7. Use Preview to view a more accurate representation, as the WYSIWYG may not render the spacing correctly in the editor.

For more information, see the Flare help for Adding Notes or Comments Between List Items.

NOTE There is a TRSS issue with the style sheet where the <p> and <li> font sizes are different.
