Importing a Flare Project from TFS

Summary:

To import a Flare project from the Team Foundation Server.

Full Description:

  1. Open MadCap Flare.
  2. Select New Project from the File menu, and then select Import Project.
  3. Enter http://tfsapp-na2-03q:8080/tfs/HRT into the Server box.
  4. Next to the Team Project box, click Browse, select Docs, and then click OK.

    5. NOTE If you encounter an error, contact your System Administrator to add you in TFS.

  5. Click Next.
  6. Click Browse, and then navigate to the desired help project:
    • CompConnect Project: $/Docs/CompConnect
    • TRSS Project: $/Docs/TRSS1
    • HRPortals Project: $/Docs/HRPortals
    • Standards: $/Docs/Standards
    • etc.
  7. Select the project file (*.flprj), and then click OK.
  8. Click Next.
  9. Leave the project name, but change the value in the Project Folder box to a location on your C: drive, such as C:\Workspace.

    10. NOTE If you are re-importing a Flare project, then you must import the project to a different directory or folder on your C:\ drive, for the import to work.

  10. Click Finish.
