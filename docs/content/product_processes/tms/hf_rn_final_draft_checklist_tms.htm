Final Draft Checklist for TMS Hotfix Release Notes

Use this checklist to perform periodic clean-up tasks on the TMS Hotfix Release Notes before sending them to the product owner for final approval.

Date Assignee Tasks
Thurs (2nd week) TCOMM During the draft creation, keep the number signs in the red highlights at the beginning of the HF template. The product owner replaces the symbols with the build number when generating the final copy.
Remove the red highlights in the final version of the RN.
Arrange the tables alphabetically and the tickets in each table in ascending numerical order.
Delete empty tables in the final version of the RN.
Remove all extra lines in the tables.
When rows of a table are not properly spanning multiple pages, such as when a table heading is on a page by itself, break the table at an appropriate row by inserting a page break at the beginning of the row.
Set all summaries and resolutions to table text.
Set all ticket numbers to hyperlink.
Remove all colors from tables.
Remove all text highlights.
Make sure all tables cells are aligned correctly.
Make sure that all entries are in the correct heading modules.
