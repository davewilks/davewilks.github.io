Using Image Maps

Summary:

Image maps are used in the online output for the homepage orientation topic and the landing page orientation topic. Popup windows appear with component information, as the user clicks each component number on the image.

Pre-requisites:

The component text that appears in popup windows must be created first. These "mini" topics are saved in Getting Started. They contain a snippet for the component description and the component image. Key features are listed, when available.

The homepage and landing page images must be saved with the stylized number: already applied. The stylized numbers are applied to the image in Snagit.

Solution:

To add or edit the image map numbered items:

  1. Right-click the image in the homepg_orientation.htm file and then select Image Map.
  2. In the local toolbar select the shape you want to use for the image map. Most of the time you will use a circle shape. On the MegaMenu, Content Slider, and Content Tiles, a square shape was used.
  3. On the image, draw the shape over the number, or select the current shape for the number.
  4. Click in the Link line of the dialog box. A navigation dialog box opens. Navigate to the topic that is to appear when the use clicks that number. For example, #2 is search, so select search.htm in the Getting Started folder. No 3 is the MegaMenu, so select the homepg_megamenu.htm file in the Getting Started folder, and so forth.
  5. Select Popup Window for the Target Frame.
  6. Click OK. The path to the popup window topic appears In the Link properties.
  7. Optional. Enter a Tab number (that corresponds with the number on the image) to allow users to tab through the popups on the image.

Use this same procedures for the landingpg_orientation.htm topic.
