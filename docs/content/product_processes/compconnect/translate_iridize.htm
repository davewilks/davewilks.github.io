Translating Iridize Tutorials

Summary:

TComm is responsible for translating Iridize content.

Full Description:

Many steps are also available from Iridize help. As you go through the process, take a look at their readmes and other guidance.

To translate Iridize:

  1. Create a ticket for the translation work. See this example.
  2. Use IE to open the Translations Confluence page. With IE you can edit the spreadsheet from the page. In browsers other than IE, access the Attachments and work with it there.
  3. Open the translation log and keep it updated throughout this process.
  4. Export from Iridize: Customize > Export.
  5. Save to a folder associated with a release and date (e.g., R1.1|Iridize).
  6. Rename the zip file to include the date: iridize_export_04152016.
  7. Extract all contents.
  8. Navigate to ..\testing\--.
  9. Zip up the folder and send to translator. Make sure you rename it first, -- doesn't work well.
  10. When you get the files back, add them to ..\import\en-gb (or whatever language code).
  11. Zip up the Import directory. There are very specific instructions in the README.html.
  12. Upload it to Iridize.
