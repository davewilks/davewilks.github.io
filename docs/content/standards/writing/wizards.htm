Wizards

Summary:

Only use the word "wizard" in the help topics when the interface includes the word.

Full Description:

If the Talent|REWARD interface uses the word "wizard", in some form, then you can use that word in the help topics. If the interface does not say the word "wizard" then do not use it in the help topics. Use instead the menu option as the compound noun or add the word page, window, dialog box, etc., as appropriate.

Correct: Use Period Setup to manage talent.

Incorrect: Use the Period Setup wizard to manage talent.
