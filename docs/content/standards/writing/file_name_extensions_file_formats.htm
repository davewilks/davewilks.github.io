File Name Extensions

Summary:

The MMOS standard is somewhat confusing and hard to find, so TCOMM is using a different, simpler standard.

Full Description:

  • For “file types” or “files” use upper case without a period. (This applies to both abbreviations and other types, including CSV, XML, XLIFF, PNG, JPG, JPEG, TIFF, BMP, XLS, DOC, HTML, RTF, and PDF.
  • For “file formats” or “formats” use lower case with a period. (.pdf)
  • If the UI specifically exports to a “Word” or “Excel” file, try to use file types because these are not technically formats, but if you must also include formats, you can mix the UC/LC styles. (PDF, MHTML (web archive), Excel, or Word)
  • Occasionally it is beneficial to spell out the abbreviation/acronym (MMOS has a helpful list on page 216.)

Examples:

Export the displayed report to an Excel, Word, CSV, or XML file.

Send the report to a printer or create an HTML-powered PDF document.

Select an export file type: PDF, MHTML (web archive), Excel, or Word.

Supported file formats: .doc, .docx, .pdf, .xls, .xlsx, .txt, .rtf, and .csv

Export an image of the chart to a JPEG or PDF file.

Export an image of the chart to .jpeg or .pdf file format.
