Then vs. and Then

Summary:

A common step in procedure topics is instructing the user to do one thing, and then another. One thing comes first and is immediately followed by a second. This is our standard:

Click File, and then select New.

Although there are other ways to say this, the grammatical reasons to avoid writing this way are shown below:

  • Click File, then select New. This is grammatically incorrect because it is a comma splice. The comma tries to hold together two independent clauses and it needs a coordinating conjunction to help out, such as and.
  • Click File and click New. This is OK, but the "and" indicates a simultaneous action.
  • Click File then click New. No comma splice here, however you're using then as a conjunction, which it isn't. It is a conjunctive adverb, which brings up all sorts of problems including semicolons and multiple commas.

The take away? Write: Click something, and then select something.
