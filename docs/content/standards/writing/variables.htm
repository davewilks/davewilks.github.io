Variables

Summary:

Use variables in Flare for common terms or terms that might change. Examples of variables include:

  • Company Name
  • Application Name
  • Module Name (such as Tenant Admin)
  • Telephone numbers
  • Year
  • Other information that you might use occasionally and need to remain consistent throughout the project
  • Unknown terms or terms likely to change. For example, button names that might be uncertain, screen or page names that aren't settled on, or terms that could be different in different versions of the applications.

NOTE Variables cannot contain formatting! Use a snippet as a workaround.

Tech Comm defines each project's variables at the project level. Comments about each variable and usage are listed with the variable. Variables are in the Variables folder, located in the Project Organizer.

To use a variable:
  1. Begin typing the word.
  2. The Auto Suggestion box displays.
  3. Press Enter to insert the variable.
  4. If the Auto Suggest box does not display, click Insert, and then select Variable.
  5. Select the variable name from the variable set.
