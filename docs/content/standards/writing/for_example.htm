For Example

Summary:

Describes the use of "for example".

Full Description:

When using For Example in a brief example, use an em dash (—) and a comma:

Select a unique non-numeric comparator from the drop-down list—for example, Average or Median.

When using For Example in a more lengthy example, start a new sentence:

These shortcuts provide limited versions of the functionality located elsewhere in the module. For example, if you do not need to view the Functional Permission Groups displayed on the Roles page, you can make quick user role modifications from the Actions menu.
