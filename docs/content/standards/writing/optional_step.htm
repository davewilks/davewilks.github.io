Optional Step

Summary:

When you have a task in which there are specific items that you must complete to accomplish it and you have other items that are optional. When a step in a task is considered optional, write the word Optional with a period at the beginning of the step.

  1. Optional. Complete the Description field.

Full Description:

Tell the user how to perform the task based on what is required, everything else is optional.

For example, to set up a user, you need a first name, last name, and email address. Other fields include type, description, location, and favorite Phil Collins song. The required fields include *, so it’s clear to the user that they’re required.

To create a new user:

  1. Enter the first and last name of the user. (recognizing that even our most basic users are not PC novices)
  2. Enter the email address.
  3. Optional. Complete the following fields… or whatever) complete the following fields to further define the user:
    • Type
    • Description
    • Location
    • Favorite Phil Collins Song
  4. Click Add.

You might have a case where the fields aren’t required to advance without an error, but they are required to perform a task. Treat them the same way:

To create a new administrator user:

  1. Enter the first and last name of the user. (recognizing that even our most basic users are not PC novices)
  2. Enter the email address.
  3. Select Administrator from the Type list.
  4. Optional. Complete the following fields… or whatever) complete the following fields to further define the user:
    • Description
    • Location
    • Favorite Phil Collins Song
  5. Click Add.
