eLearning Terminology

Summary:

Defines terms used to describe eLearning tutorials.

Full Description:

Walkthrough user guides done in Iridize and incorporated into the Workforce Analytics (WAPS) application are referred to as tutorials.

Application vs. System - application and page are the preferred "techie" references in WAPS.

Storyboard - an outline of an eLearning course or tutorial that is prepared before the development process begins. It helps identify the text, images, visuals, and other elements to be used in the tutorial.

Tutorial - A computer-based instruction (CBI) technique in which information is presented onscreen in a step-by-step manner. Generally, tutorials are designed to teach the user a specific skill or set of skills. They are used regularly to teach users how to use a new software program.

Gamification - Techniques that turn eLearning content into an interactive and engaging experience for learners. The techniques present information to learners in a fun, unique way and can include gaming elements such as rewards, levels, titles, and positive feedback.

User - avoid this word instead refer to learner or participant or simply use "you".

Types of tutorials

  • Task-based tutorial
  • Overview tutorial
  • Descriptive tutorial (one-page tip)
