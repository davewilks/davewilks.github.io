Multiple Lists on a Single Page

Summary:

Use Flare's drop-down text feature for lists when multiple lists appear on a single page. This allows the reader to focus on one task at a time. Use Flare's expanding text feature on field definitions, particularly in TRSS reference documentation.

Full Description:

Ordered Lists

Formatting for drop-down lists follows the standards mentioned in Lists, with one exception:

Do not create an introductory stem with an infinitive and a colon. Instead, introduce the list by its action and use this phrase as the drop-down header.

Example

Unordered Lists

For generic unordered lists, use Flare's expanding text feature:

  • Use bullets to set off the list.
  • Use at least two list items in an unordered (bulleted) list.

    • For unordered lists of on-screen elements, such as field labels, text boxes, buttons, and icons, use expanding text for the descriptions. The names should remain visible, without punctuation, so that the reader can scan them quickly and click only those that require explanation.

Example

For bulleted lists of terms and their definitions, bold the term followed by an unbolded em dash (—) with no spaces before or after, with the definition immediately following. Capitalize the first word of the definition and use a hanging indent flush with the text.
