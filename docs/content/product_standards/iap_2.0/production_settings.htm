Camtasia Production

Summary:

Defines the settings for producing a video from Camtasia Studio.

Full Description:

IAP

This is IAP specific, other projects have different settings that are or will be described here.

Get the IAP project

The IAP project is source controlled in a Flare project.

  1. Check out the zip file containing the project to update.
  2. Close Flare and don't check in
  3. Go to the Content directory in your workspace and copy the zip file to another directory.
  4. Unzip the project and open it in Camtasia.
Produce the IAP project
  1. With the project open, select Produce and Share in the toolbar.
  2. If you have any interaction in the video, such as a TOC or even a Next button, select Custom Production Settings.
  3. Select the MP4 Smart Player.
  4. On the Smart Player Controller tab set options for the controller appearance and whether the video pauses at the beginning.
  5. On the Size tab set the Embed size and the Video size. The Video size is the size you specified for the Editing Dimensions in Camtasia.
  6. On the Video Settings tab make sure the Encoding mode Quality is set to at least 70%.
  7. On the Options tab, select any options you want enabled.
  8. Click Next.
  9. On the Video Options page, click HTML options, and enter the title of the video.
  10. Click Next.
  11. Specify the output file name (that will kick it off) and the folder for your generated output. For example:
    • edit properties.html
    • C:\IAP\videos\editing_properties\Production saves the files used to view the video in a Production folder, separate from the working files
    • You can specify the review drive location and produce the video directly to that drive; then give the link to reviewers. trciis-na2-201d:82/help/IAP/videos/folder/video_name.html
Saving the IAP project

When you're ready to check it in:

  1. In Camtasia, go to File>Save project as zip.
  2. Copy the zip file back to the Flare directory in your workspace.
  3. Check it in.
