Camtasia Recording Settings

Summary:

Defines the optimal settings for laptop resolution, editing dimensions, recording size, and production output in Camtasia Studio.

Full Description:

TIP Before recording, set your Task Bar to auto-hide, to exclude it from the video footage. (However, you can also crop it out of screen recordings in Camtasia.)

  Camtasia
Application Screen Resolution Editing Dimensions Recording Size Production Size

1366 x 768

This is the recommended laptop resolution.

Run the application on the laptop and do the screen recording here.

Use F11 to run the browser in full screen mode for recording, if desired.

 

1280 x 720p is the recommended standard if you don't know the exact device your viewers will be using.

Select the size for the final video. BUT, chose a size that is smaller than the recording size, IF you are using the Smart Focus Pan & Zoom functions on screen recordings (which you probably will). This gives the Zoom function some room to "zoom".

Custom

A dimension appears when you select the Record function. Generally, you will record at full screen, but the actual recording size will vary from screen to screen. It gets set when you set the recording region.

Smart Player up to 720p

This output enables hotspot usage, and other interactivity. It displays a progress bar for the user.
