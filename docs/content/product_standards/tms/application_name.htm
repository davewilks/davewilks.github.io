Talent Management Software Product Name

Summary:

The correct capitalization and formatting of the product name is Talent Management Software, formally known as Talent|REWARD. Do not use any abbreviations or acronyms for this three-word name. The variables and snippets in Flare are up to date for this product name. Please use the variable where possible except in snippets where Flare will only allow snippets.

The customer-facing name for the CompConnect Tenant Admin portion of Talent Management Software is Admin Home (in the CompConnect Help Center topics). There is a variable in Flare for Administration.

Solution:

  1. In TMS, use the Application_Long snippet. We have two snippets: Application_Long and Application_Short because Talent|REWARD was abbreviated to TRSS. In the new branding, we have Talent Management Software and it does not have an acronym but I left the Application_Short snippet in the files in case we change it to TMS someday.
  2. In CoCo, use the Administration variable by beginning to type the word. The Auto Suggestion box displays.
    1. Press Enter to insert the variable.
    2. If the box does not display, select Variable from the Insert menu, and then select Tenant Admin (module) from the UI variable
