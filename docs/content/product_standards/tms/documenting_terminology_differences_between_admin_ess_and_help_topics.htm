Documenting Terminology for Admin, ESS, and in Help Topics

Summary:

Describes terminology discrepancies due to custom field labels, terminology substitutions, and ESS changes and the documenting process for TCOMM.

Full Description:

While documenting the Talent|REWARD Admin areas, you may find discrepancies between what the logged-in user sees and what the help topics define.

These variances are due to the following modifications to the base Admin site:

  • TW customizes a client’s initial installation.
  • Clients use the Custom Field Labels option in Admin that impacts elements in Admin and ESS
  • Clients use the Terminology Substitution option in Admin that impacts elements in Admin and ESS.
  • ESS users can change some elements on their page views (headings and instructions).

As a writer, please evaluate the differences that you may encounter. Perform the evaluation on a new client build with the latest Sales demo data.

NOTE Small differences probably don’t matter such as % or Percent for a custom value.

Discuss these customizations and terminology substitutions with your peer and PA/CD reviewers. Share changes with the TComm team using the table at the bottom of this section.

The following sections explain how to view the differences between Admin, ESS, and help topics.

Custom Field Labels

An administrator can change field labels in the People > Personal Information, Employment Information, and Compensation Information pages to better reflect a client’s working environment.

See the System Administration > Configuration > Custom Field Labels topic. The Custom Field Labels page displays all of the possible values to change and the custom values. You can search and use categories to filter the list.

Use Flare to search for instances of the custom field labels in help topics.

Terminology Substitution

An administrator can change terminology in Admin and ESS to better reflect a client’s working environment.

See the System Administration > Configuration > Terminology Substitution topic. The Terminology Substitution page shows all of the terminology substitutions that have been made.

Use Flare to search for instances of the terminology substitutions in the help topics.

ESS Changes to Page Headers and Instructions

There is no topic for this item as ESS does not have help topics yet.

Employees can customize page headings and descriptions in their own environment using the gear button next to a page title. The button opens the Customize Page Headings and Instructions page.

The administrator would have to log in as the employee to see the employee’s page customization and then compare the changes to a base ESS site.
